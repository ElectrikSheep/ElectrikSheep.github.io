    Game Experience
    Whenever I have the occasion I try to find some time in my busy dev schedule and just try to make some things :
    • Space Walk :
      First I was merely helping helping a friend to write some script. But then I took over the role of developer for his artistic vision.
      It is a game targeting mobile platforms, using the gyroscope to control the main character. The goal is to run as far as the user can, while avoiding foes.
      A version for Keyboard and mouse is available by clicking on the title or here.
    • Phone Of The Dead :
      The company Digicave happened to have a few Zombies 3D models laying around and saw the opportunity to re-use part of the gyroscopic code I wrote for Space-Walk, while playing with the concept of re-usable objects, avoiding to Instantiate of Destroy anything in the code and just re-cycle everything (green).
      The Facebook version of the game is available here.
      And slowly but surely I am fine tuning the iOS version so I can publish it.
