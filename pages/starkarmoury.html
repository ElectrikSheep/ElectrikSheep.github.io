Stark Armoury at Tesco

An iOS/Android project built using Unity3D. My role for this project Developer.

Stark Armoury at Tesco was an application made to advertise for the movie Ironman 3 and draw customers to Tesco supermarket. We had to work with Marvel to get and use their assets in the application following their guidelines.

Concept and Development

Once the application was downloaded, the user would be allowed to interact with only one Ironman Suit, the Mark II. They could choose between 3 experiences.
The project was mainly made using Unity and C#.

An augmented reality experience triggered from a flyer disponible in store or printable from the app. It would show Ironman standing on top of the flyer, allowing the user to see the armoured suit in their own home/environement.

A photoshoot experience. The user take a picture of his face wich would then appear inside a picture of the selected armour. The code to crop the face is a plugin I made using native iOS/Android code to use within the Unity .

A gyro based experience, the user would see the Ironman arms on top of a video feed of the device camera. With a simple touch it would fire a blaster in the environment and leave a mark on it. The code in this experience is very flexible as working with big company can bring a lot of changes during the conception of a project.

In order to access the same experiences with the yet to be unlocked suits, the user had to scan signs at Tesco.
